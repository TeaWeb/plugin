## 跟TeaWeb主线程交互说明

### 接收
* Start - 启动插件
* Stop - 停止插件
* Reload - 刷新插件
* ReloadWidget - 刷新Widget
* ReloadChart - 刷新Chart
* ReloadApp - 刷新单个App
* FilterRequest - 过滤HTTP请求
* FilterResponse - 过滤HTTP响应 

### 发送
* Register - 注册插件
* ReloadChart - 刷新Chart
* ReloadWidget - 刷新Widget
* ReloadApps - 刷新插件相关的所有Apps
* ReloadApp - 刷新单个App
* FilterRequest - 过滤HTTP请求结果
* FilterResponse - 过滤HTTP响应结果
